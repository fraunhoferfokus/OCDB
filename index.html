Open City Database SE - Quick and easy deployable cities related data service with read/write access to user generated content, points of interest and social interactions (likes, comments, ratings, check-ins).

Get in touch now!

To get you OCDB interested folks from zero to hero we have an API console established enabling you to play around with the service. So let's start...

  1. To Register at the public endpoint.
  2. Login there. You will be provided with an access token.
  3. Use the access token in the API console for your fun. But be kind.

Notice: there is no warranty on availability of the public endpoint provided here. Set up your own. It's not that difficult ;)

Playground

Have a look at the simple jsfiddle snippet and be invited to tweak the sample code and learn how easy the OCDB service can be used within apps.

Documentation

The API is fully documented and provided via swagger description file. The most recent version is available within the API console (see Get in touch now!). Current version of the API is v1. For information how to setup and deployment please have a look on the README.
